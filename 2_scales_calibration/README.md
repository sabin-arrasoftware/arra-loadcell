# Load Cell GUI

## Description
This Python code provides a graphical user interface (GUI) for displaying the output values from load cells. 

## Prerequisites
- Python 3.x
- Tkinter library (included in Python standard library)
- Arduino
- 2 load cells
- 2 HX711 modules
- breadboard
- USB cable for Arduino
- connecting wires
